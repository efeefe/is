### Description

An open database collecting and making available different data about objects, in order to aid reuse (repair, upcycling, secondary market).

Examples of data do be listed:

* Unique Identifier
* Manufacturer
* Versions
* Materials
* Service manuals
* Parts and tools
* Repair tutorials
* Examples of reuse
